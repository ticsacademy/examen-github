Examen Git y Github

Alumn@: Cesar Omar Perez Hdz

  1. Que es un sistema de control de versiones?

    Se podria decir que es un sistema que nos permite controlar todos los cambios realizados a un documento

  2. Cual es la diferencia entre git y github?

    git se utiliza localmente, y github se utiliza ya cuando se requiere colaborar con mas personas de manera remota.

  3. Cómo creo una rama en git?

    git checkout -b [ nombre que va a tener tu rama ], o git branch [ nombre que va a tener tu rama ]

  4. Cómo clono un repositorio?

    vas al repositorio que quieres clonar, le das al boton de [ Clone or download ] y copias esa url, en consola tecleas [ git clone "url copiada" ] y tendas la carpeta con los archivos que contiene el repositorio

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    primero se realiza un git fetch, despues un git pull

  6. Cómo puedo gener un tag?

    despues de hacer un commit se escribe en consola el comando de git [ git tag "numero de la version" -m "mensage del tag" ]

  7. Cómo hago una mezcla de ramas?

    primero me posisiono en la rama destino y procedo a ejecutar el comando [ git merge "rama-a-combinar" ]

  8. Cómo puedo moverme a un commit especifico?

    se puede hacer tecleando [ git checkout "id-del-commit-a-movernos" ]

  9. Cómo puedo borrar una rama?

    aqui hay dos formas de realizar la eliminación, la primera es antes asegurarnos de que la rama haya sido combinada [ git branch -d nombre-de-la-rama ], la otra es forzandola y sin combinar [ git branch -D nombre-de-la-rama ]

  10. Cómo puedo borrar un tag?

    se ejecutara [ git tag -d "nombre del tag" ]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    se ejecuta el comando git status, para ver que esta añadido a la zona de espera ( se podria decir ) y que no se ha agregado

  12. Para que sirve el comando git remote -v?

    te permite ver a que repositorio remoto se puede hacer push

  13. Cómo le digo a git cual es mi email y correo?

    con el comando [ git config --global user.name "Nombre" ] es para decirle a git que ese es tu nombre, con el comando [ git confi --global user.email "tu correo" ] es para decirle a git que ese es mi correo, estos comandos se ejecutaran de manera global

  14. Cuales son los pasos para conectar git con github?

    lo primero es generar una llave SSH, eso se genera con el siguiente comando [ ssh-keygen -t rsa -b 4096 -C "tu-email@example.com" ], despues ejecutar el comando [ ssh-add ~/.ssh/id_rsa ], para agregarlo y que el sistema de git lo reconozca, despues se ejecuta [ cat ~/.ssh/id_rsa.pub ] para que nos muestre la llave, copiamos lo que nos aparece, ahora nos vamos a github.com y nos vamos al apartado de Settings , buscamos la opcion de SSH and GPG keys, le damos al boton de NEW SSH KEY y pegamos todo lo que copiamos anteriormente y con eso ya quedaria, solo seria crear un repositorio nuevo y seguir los pasos que se indican.

  15. Cómo puedo crear un alias en git?

    para hacer un alias, se teclea el comando [ git config --global alias."nombre del alias" "comando de git" ].
