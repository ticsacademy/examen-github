  1. Que es un sistema de control de versiones?

    Es un sistema que nos ayuda al registro de los cambios realizados en un archivo o un conjunto de archivos en un tiempo predeterminado, de modo que con este podamos recuperar versiones específicas más adelante.

  2. Cual es la diferencia entre git y github?

    Git es la herramienta que nos permite manipular los repositorios en la terminal, mientras que Github es la plataforma de hosting para trabajar con los proyectos remotamente.

  3. Cómo creo una rama en git?

    Usando el comando git branch Nombre_de_la_Rama

  4. Cómo clono un repositorio?

    Usando el comando git clone Direccion_del_repositorio

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    Usando el comando git pull o git pull origin Nombre_de_la_rama

  6. Cómo puedo gener un tag?

    Usando el comando git tag Nombre_del_tag

  7. Cómo hago una mezcla de ramas?

    Usando el comando git merge Nombre de la rama

  8. Cómo puedo moverme a un commit especifico?

    Usando el comando git checkout nombre_del_commit

  9. Cómo puedo borrar una rama?

    Usanso el comando git branch -d Nombre_de_la_Rama
