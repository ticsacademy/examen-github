  1. Que es un sistema de control de versiones?

    Es el sistema que ayuda a mantener actualizado un archivo en tiempo real, así como almacena los diversos cambios realizados del mismo, permitiendo volver a las diferentes versiones facil y rapidamente.

  2. Cual es la diferencia entre git y github?

    Git se utiliza para trabajar de manera local. En GitHub se te permite colaborar con otros programadores, así como compartir tus códigos. Es como una red social.

  3. Cómo creo una rama en git?

    git branch [nombre-para-la-rama]

  4. Cómo clono un repositorio?

    git clone [direccion-del-repositorio]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    git pull origin master

  6. Cómo puedo generar un tag?

    git tag v1.0.0 -m "mensaje"

  7. Cómo hago una mezcla de ramas?

    git checkout [rama-a-combinar-1]

    git merge [rama-a-combinar-2]
