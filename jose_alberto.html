  1. Que es un sistema de control de versiones?

    [Es un sistema que nos ayuda a tener un mejor control sobre aquellos cambios que se le vayen realizando algun proyecto en espesifico, de igual manera es muy util a la hora de compartir el codigo y no estarlo pasando por usb, whatsapp, correo, etc. otra de las ventajas es estar regresando entre versiones anteriores por si en una actual se tuviera algun problema.]

  2. Cual es la diferencia entre git y github?

    [git es el sistema de control de versiones. github (red social de los programadores) es una plataforma en la cual podemos subir los repositorios en la nube para poder conpartirlos con otras personas.]

  3. Cómo creo una rama en git?

    [git branch "nombre de la rama"]

  4. Cómo clono un repositorio?

    [git clone "ruta del repositorio"]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [git pull]

  6. Cómo puedo gener un tag?

    [git tag v1.0.0 -m "mi primer tag"]

  7. Cómo hago una mezcla de ramas?

    [git merge "nombre de la rama que quiere combinar"]

  8. Cómo puedo moverme a un commit especifico?

    [Primero teclea git log Segundo copia la serie de numeros y letrasque estan despues de donde dice commit por ultimo teclea git checkout "poner la serie de numeros y letras que copio"]

  9. Cómo puedo borrar una rama?

    [git branch -d "nombre de la rama"]

  10. Cómo puedo borrar un tag?

    [git tag -d "nombre del tag o version del tag"]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git status]
